# AR/VR依赖库

本目录包含AR/VR功能所需的JavaScript库文件。

## 文件说明

- aframe.min.js: A-Frame库，用于构建VR场景
- ar.js: AR.js库，用于实现AR功能
- jquery.min.js: jQuery库，用于简化DOM操作

## 使用方法

在需要AR/VR功能的页面头部引入这些库：

```html
<!-- VR依赖：A-Frame库 -->
<script src="./assets/lib/aframe.min.js"></script>

<!-- AR依赖：AR.js库 -->
<script src="./assets/lib/ar.js"></script>

<!-- 通用依赖：jQuery -->
<script src="./assets/lib/jquery.min.js"></script>
```
