# AR/VR校园导览系统（入门级方案）

## 项目简介
本项目是一个基于AR/VR技术的校园导览系统，支持全景漫游和AR标记识别，纯前端实现，无需后端，易出效果。

## 技术栈
- 前端：HTML5, CSS3, JavaScript, A-Frame, AR.js
- 工具：HBuilderX, VSCode

## 项目结构
```
ar-campus-guide/
├─ index.html              // 核心首页（导航入口）
├─ ar-navigation.html        // AR导航页面
├─ vr-panorama.html        // VR全景漫游页面
├─ assets/               // 资源文件夹
│  ├─ panorama/          // 全景图文件夹
│  ├─ marker/            // AR标记图文件夹
│  └─ info/              // 点位介绍图/文字
└─ css/                 // 样式文件夹
   └─ global.css          // 全局样式（适配移动端）
```

注意：以下文件为冗余文件，可以忽略：
- ar-page.html (旧版AR页面)
- vr-page.html (旧版VR页面)
- 3d-page.html (进阶级方案文件)
- 3D_README.md (进阶级方案文档)
- manifest.json (HBuilderX应用配置)

## 使用步骤
1. 启动项目：
   - 双击运行start.bat
   - 在浏览器中访问 http://localhost:8080

2. 准备资源：
   - 将360°全景图放入assets/panorama文件夹
   - 将AR标记图和pattern文件放入assets/marker文件夹

3. 使用功能：
   - 首页：选择点位，点击AR导航或VR全景按钮
   - AR导航：将摄像头对准标记图，查看导航信息
   - VR全景：拖拽旋转视角，点击导航点切换场景

## 功能特点
1. 首页导航：
   - 顶部居中显示标题"校园AR/VR导览"
   - 下方是3个点位卡片（校门、主教学楼、图书馆）
   - 卡片样式：白色背景+轻微阴影
   - 每个卡片包含点位名称、简介，底部有两个彩色按钮（AR导航蓝色、VR漫游橙色）
   - 适配效果：在手机浏览器中打开，卡片宽度占满屏幕，按钮横向排列，无横向滚动条

2. VR全景漫游：
   - 全屏显示360°全景图
   - 视角可通过鼠标拖拽（PC）/手指滑动（手机）旋转
   - 场景中悬浮导航点，点击可切换不同点位
   - 支持自动旋转和重置视角

3. AR导航：
   - 调用手机摄像头，全屏显示摄像头画面
   - 识别AR标记后显示点位信息
   - 提供语音导航和地图导航功能

## 注意事项
- 确保全景图和标记图资源已放入正确文件夹
- AR标记图建议打印在A4纸上，边缘留白2cm
- VR全景漫游需要支持WebGL的浏览器
- 建议在WiFi环境下使用以节省流量
