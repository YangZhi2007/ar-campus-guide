# AR校园导览应用

## 项目简介
AR校园导览应用(ar-campus-guide)是一款基于增强现实技术的校园导航应用，帮助新生和访客更好地了解和导航校园。

## 技术栈
- 前端框架：MUI
- 开发平台：支持Android、iPhone和iPad

## 项目结构
- css/: 存放样式文件
- js/: 存放JavaScript文件
- fonts/: 存放字体文件
- index.html: 应用入口页面
- manifest.json: 应用配置文件

## 开发指南
1. 克隆项目到本地
2. 使用支持HTML5的浏览器打开index.html
3. 或使用HBuilderX等工具进行开发和调试
