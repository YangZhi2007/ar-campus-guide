## 图片使用说明

1. 请按照上述列表从百度搜索并下载相应的图片
2. 将图片保存到images文件夹中，并使用指定的文件名
3. 确保图片尺寸符合要求，以保证显示效果
4. 图片格式建议使用PNG格式（图标）和JPG格式（照片）

## 注意事项

- 请确保下载的图片不侵犯版权
- 建议使用简约风格的图标，以保持界面一致性
- 轮播图图片建议选择高清、色彩鲜明的校园风景照片
